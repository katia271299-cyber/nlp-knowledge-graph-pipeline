

# Pipeline NLP multi-flux RSS

Ce projet implémente un pipeline NLP permettant de collecter des flux RSS,
nettoyer les articles, générer des résumés automatiques et extraire des mots-clés
et entités nommées.

## Fonctionnalités
- Collecte de flux RSS multi-sources
- Nettoyage HTML et prétraitement du texte
- Résumé automatique avec LexRank
- Extraction d'entités et mots-clés (spaCy)
- Sortie structurée des résultats

## Technologies
Python, feedparser, spaCy, NLTK, sumy
